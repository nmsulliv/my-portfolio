C++ Infinite Natural Number

This is a C++ program which computes addition, multiplication, or factorials no matter how large the result is. For example: 50! cannot normally be computed with type long int, but this program is able to calculate and print the result - 30414093201713378043612608166064768844377641568960512000000000000.

This project taught me how to create an arbitrary precision calculator, and the importance of arbitrary precision calculations to avoid overflow.

MIPS Base Converter

This is a MIPS program which reads a string of exactly 10 characters from user input. It will then output the conversion of the string in base 30. However, if the string contains characters outside of '0-9', 'a-t' or 'A-T', it will print out invalid input.

This program taught me the basics of assembly programming, how to work with registers, and how to use QTSpim. Originally I thought this program seemed simple, but I quickly learned how intricate assembly programming is, and how important it is to efficiently manage memory.

C++ Benchmark Project

This is a C++ program which computes the execution time for billions of CPU, memory, and hard drive operations.

Through this project I learned how to write benchmark tests, and the importance of SPEC Benchmarks. I also learned how to write binary files - which I think is pretty cool!

Java Ten-Pin Bowling Calculator

This program takes in an integer array of a sequence of throws for one game of Ten-Pin Bowling, and produces the score for the game.

There is currently no GUI, but the purpose of this coding exercise was to learn about JUnit testing. This project taught me how to use the JUnit testing framework, as well as how to code in Java.
